# Voice Assistant using Python

This project is a simple voice assistant built using Python.
It listens to user voice commands and performs actions like
telling the current time, opening Google, and searching Wikipedia.

## Features
- Speech Recognition
- Text to Speech
- Time Query
- Google Search
- Wikipedia Search

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API

## How to Run
1. Install required libraries
2. Run the file using:
   python voice_assistant.py
